Halo, Saya Aldiano Prayugo

Halo, saya Aldiano Prayugo, saya adalah murid di Santa Laurensia Suvarna Sutera. Saya menyukai pelajaran Math karena dapat menantang dan melatih logika saya. Selain itu saya juga suka belajar Bahasa Indonesia karena sangat menyenangkan dan gurunya saat mengajar mudah dimengerti. Pelajaran PE menjadi pelajaran paling favorit saya karena semua materinya sangat menyenangkan. Saya telah bersekolah di Santa Laurensia sejak kelas 3.

Saat ini saya sedang fokus untuk meningkatkan nilai di semua mata pelajaran. Saya berusaha belajar lebih giat agar dapat mencapai hasil yang lebih baik. Dengan melatih agar saya lebih teliti, saya berharap bisa membanggakan orang tua saya. Saya percaya bahwa usaha tidak akan mengkhianati hasil. Karena itu, saya terus berkomitmen untuk belajar dengan sungguh-sungguh setiap hari.
